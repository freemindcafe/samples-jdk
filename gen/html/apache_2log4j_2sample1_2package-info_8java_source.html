7  * </p>
8  * <p>
9  * TRACE > DEBUG > INFO > WARN > ERROR > FATAL > OFF
10  * </p>
11  * <p>
12  * For this example the log file is generated with in the package directory.
13  * </p>
14  * <ul>
15  * <li>FooBeans's appender has been set and it's additivity is false. Hence it's log messages will go in FileLog appender only and not in stdout appender.</li>
16  * <li>BarBeans's appender has been set and additivity is false. Hence it's log messages will go in FileLog appender only and not in stdout appender.</li>
17  * <li>App does not have any appender and additivity is by default true. It's messages will go to rootLogger's appenders stdout and FileLog.</li>
18  * </ul>
